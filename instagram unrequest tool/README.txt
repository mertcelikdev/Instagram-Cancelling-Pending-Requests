
# Instagram Pending Follow Requests Canceller (2FA destekli - FIXED)

Bu versiyon, 2FA kodunu girmen için doğru yerde bekleme yapacak şekilde düzeltilmiştir.

## Kullanım
1. Zip'i ayıkla.
2. run.bat'a çift tıkla.
3. Instagram giriş ekranında kullanıcı adı + şifreni gir.
4. Eğer 2FA açıksa sana gelen 6 haneli kodu tarayıcı ekranına yaz.
5. Script 2 dakika boyunca seni bekler. Kodu doğruladıktan sonra otomatik devam eder.
6. Sonuçlar `results_cancel_follow_requests.csv` dosyasında olur.
